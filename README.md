# Yen-Sheng's Dynamic Grade System management platform

> **PROJECT STATUS: DISCONTINUED**
> This project has been canceled by the upstream sponsor. Development is no longer active.
> The code is made publicly available under the MIT License for anyone who may find it useful.

A secure, privacy-focused grade management platform for professor Chen Yen-Sheng's courses at Taipei Tech.

## Tech Stack

- **Framework**: Nuxt 4.2.2 with Vue 3 and TypeScript
- **UI**: shadcn-vue + Tailwind CSS v4
- **Database**: PostgreSQL 18 with Drizzle ORM
- **Authentication**: Better Auth with Google OAuth
